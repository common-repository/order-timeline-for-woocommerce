The Most Advanced Order Timeline for WooCommerce. It responsive order timeline with detailed history and notes of order in the single order view page.

== Description ==

Timeline is one of the most trendy elements of a website nowadays. Order timeline for WooCommerce is a WordPress plugin that will allow you to organize WooCommerce orders. It has infinite options to organize and customize the timeline's colors, icons, themes which is an effective and professional way to beautify your WooCommerce order view without the help of professional or coding skills.

It will keep all available status history of an order like On hold, Processing, Pending Payment, Pending, Completed, Cancelled, Refunded, Failed, etc with time, date, and the note to the customer. So that customers will able to know the real update of his/her order, It's a professional way to communicate with the customer which builds trust.

###Features###

* Custom Note To Customer
* Unlimited Colors / Themes
* Clean And Compact Design
* Compitable WIth All Themes
* Different Icon For Each Status
* Custom Order Status Support
* Responsive
* Highly Customizable
* Many More..


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/order-timeline-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the "Order Timeline" submenu under the "WooCommerce" menu to configure the plugin.



== Frequently Asked Questions ==

= Can I Set Custom Note for Customer ? =
Yes, You will be able to set custom notes for customers to give an update with each status change.
